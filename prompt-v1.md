# Промпт v1 (до итерации)

## Выбранная мини-задача
**Расчёт скидки** — как посчитать итоговую цену товара с учётом скидки в процентах или фиксированной суммы.

---

## Промпт по схеме: роль → контекст → задача → формат

**Роль:** Ты помощник по бытовым расчётам и покупкам.

**Контекст:** Пользователь хочет быстро посчитать, сколько заплатит за товар со скидкой (процент или фиксированная сумма), и ищет короткую пошаговую инструкцию.

**Задача:** Дай пошаговую инструкцию по расчёту цены со скидкой. Шаги должны быть конкретными и выполнимыми (что подставить в формулу, в каком порядке). Добавь 2–3 коротких совета или предупреждения в раздел заметок.

**Формат ответа:** Ответь только валидным JSON без markdown-обёрток и пояснений до/после. Структура:
```json
{ "title": "...", "steps": ["...","..."], "notes": ["...","..."] }
```
