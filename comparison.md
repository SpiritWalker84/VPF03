# Сравнение ответов: до и после итерации (тема: расчёт скидки)

| Критерий | Ответ v1 | Ответ v2 |
|----------|----------|----------|
| **Соответствие JSON-формату** | Да | Да |
| **Полезность шагов (steps)** | Хорошая: 5 шагов (цена → тип скидки → формула % → формула ₽ → итог), с примерами. | Хорошая: 4 шага, те же формулы и пример для %, без лишних пунктов — компактнее. |
| **Лаконичность notes (до 3 тезисов)** | 3 пункта — укладывается; третий про «несколько скидок» полезен. | 2 пункта — строго в лимите, только важное (проверка и накопительные скидки). |

**Вывод:** После итерации (уточнённая роль + few-shot по расчётам) ответ v2 сохраняет полезность и формулы, сокращает шаги до необходимого минимума и укладывает notes в 2 тезиса. JSON в обоих случаях валидный. Итоговый вариант для артефакта — **response-v2.json**.
