# Расчёт скидки

Короткая инструкция: как посчитать итоговую цену товара со скидкой (в процентах или в рублях).

---

## Что сделать по шагам

1. **Возьми исходную цену товара** (до скидки).
2. **Если скидка в %:** умножь цену на (1 − процент/100).  
   Пример: 5000 ₽, скидка 20% → 5000 × 0,8 = **4000 ₽**.
3. **Если скидка в рублях:** вычти сумму скидки из цены.
4. **Результат** — сумма к оплате.

---

## Важно

- Проверь, что скидка не больше цены товара.
- Несколько скидок подряд считаются по очереди от текущей цены, а не от исходной.
