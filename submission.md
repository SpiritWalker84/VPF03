# Формат сдачи

**Тема мини-задачи:** расчёт скидки (как посчитать цену со скидкой в % или в рублях).

## Содержимое репозитория

| Файл | Назначение |
|------|------------|
| `prompt-v1.md` | Промпт до итерации (роль → контекст → задача → формат) |
| `prompt-v2.md` | Промпт после итерации (+ уточнённая роль, few-shot) |
| `run-params.md` | Параметры запуска: инструмент, модель, temperature 0.2 |
| `response-v1.json` | Ответ модели на промпт v1 |
| `response-v2.json` | Ответ модели на промпт v2 |
| `comparison.md` | Сравнение двух ответов (JSON, steps, notes) |
| `README.md` | Артефакт для пользователя: пошаговая инструкция по итоговому ответу |
